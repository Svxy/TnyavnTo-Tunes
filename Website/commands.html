/help

Displays all commands for the bot.


/kill

Kill a user with TnyavnTo.


/loop

Sets loop mode. Ex: Track, Queue


/move

Moves a song in the queue.


/nowplaying

Displays the current playing song.


/play

Plays almost any song either by link or name.


/pause

Pauses the current song.


/ping

Displays TnyavnTo's server/api ping.


/playtop

Plays a song, placing it at the top of the queue.


/queue

Displays the current songs in queue.


/remove

Removes song from the queue.


/resume

Resumes the current song.


/shuffle

Shuffles through the queue.


/skip

Skips the current song.


/stop

Stops all songs in queue.


/swap

Swaps the position of a song in queue.


/upvote

Displays links to upvote TnyavnTo.


/volume

Changes the bots volume.


/website

Displays link to TnyavnTo Tunes website.
